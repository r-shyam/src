This folder contains the Matlab program  for predicting child's engagement level in an adult-child interaction. Please refer to the following paper for more details on the proposed algorithms.

Shyam Sundar Rajagopalan, O.V. Ramana Murthy, R. Goecke and Agata Rozga, Play with Me Measuring a Childs Engagement in a Social Interaction. Proceedings of the IEEE International Conference on Automatic Face and Gesture Recognition (FG 2015), Ljubljana, Slovenia, 4-8 May 2015.

JA_UGM_HCRF.m - Stage 1 implementation.
JA_HCRF_SVM.m - Stage 2 implementation.
data - The directory containing sample features and ground truth labels.
output - The program outputs are written into this directory.
